# JadeView 火山 SDK

JadeView 火山模块用于在火山项目中调用 JadeView 的窗口、WebView、IPC、托盘、通知、对话框和系统能力。

## 安装

1. 下载模块安装包
2. 安装到火山模块目录
3. 在项目中引用 `火山.JadeView`
