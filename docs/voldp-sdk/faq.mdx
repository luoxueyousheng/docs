# 常见问题

## 哪些事件需要手动订阅

- `窗口即将关闭`
- `即将导航`
- `请求新窗口`
- `下载开始`
- `文件拖入`

## 可拦截事件怎么返回

- 返回 `1`：拦截
- 返回 `0`：放行

## IPC 推荐怎么写

使用对象方法回调,避免静态方法掣肘：

```wsv
Jade.置IPC频道回调 (本对象, "ipcChannelMessageCallback")

// IPC回调只需要在初始化前设置一次即可

```

## 异步对话框需要手动释放吗

不需要。

## 托盘事件会自动订阅吗

会。`JadeView_托盘.创建()` 会自动订阅托盘事件。

> 托盘事件回调可以点击申明的托盘类变量直接快速创建, 推荐在启动方法下申明托盘类
