# 数据类型

## JadeView_窗口选项

用途：

- 传给 `创建窗口(...)`
- 控制窗口标题、位置、大小、边框风格、主题、背景颜色等

常见字段：

- `标题`
- `左边`
- `顶边`
- `宽度`
- `高度`
- `可调整大小`
- `允许最小化`
- `允许最大化`
- `透明窗口`
- `边框风格`
- `主题`
- `背景颜色`
- `内容保护`

## JadeView_视图选项

用途：

- 传给 `创建窗口(...)`
- 配置 WebView 行为

常见字段：

- `允许全屏`
- `后台节流`
- `自动播放`

## JadeView_通知参数

用途：

- 传给 `显示通知(...)`

常见字段：

- `标题`
- `正文`
- `图标`
- `超时毫秒`
- `按钮1`
- `按钮2`
- `文本3`
- `动作`

## 回调模板

### IPC频道消息模板

用于 `置IPC频道回调(...)`，推荐使用对象方法和输出名。

### 对话框异步对象回调模板

用于异步对话框结果回调，参数为 `文本型 JSON`。
