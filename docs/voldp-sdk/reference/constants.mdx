# 常量

本页列出当前模块中常用的常量类。

## 用于窗口与外观

- `主题颜色`
- `背景材质`
- `边框风格`
- `提示框类型`

这些常量主要用于：

- `JadeView_窗口选项`
- `置窗口主题(...)`
- `置窗口材质(...)`
- `消息框(...)`

## 用于系统能力

- `热键功能键`
- `路径常量`
- `对话框属性`

这些常量主要用于：

- `注册全局热键(...)`
- `取路径(...)`
- 文件对话框属性字符串

## 用于事件订阅

- `订阅_窗口事件`
- `订阅_视图事件`
- `订阅_通知事件`
- `订阅_系统事件`

说明：

- 大部分普通事件会在 `初始化()` 时自动订阅
- 可拦截事件和 `文件拖入` 这类特殊事件需要手动订阅
- 订阅事件的回调可以直接在类申明变量那里点击闪电图标快速申明回调方法

## 用于托盘

- `托盘菜单项类型`
- `托盘事件类型`

说明：

- `托盘菜单项类型` 用于 `添加菜单项(...)`
- `托盘事件类型` 用于理解 `托盘交互事件` 的事件类型参数
