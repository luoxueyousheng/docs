# Examples

Copy-paste-ready snippets covering windows / IPC / tray / hotkeys / context menus / dialogs / drag-drop / YAML. Snippets live inside `类 启动类 <公开 基础类 = 程序类>` with the JadeView module referenced and these declared:

```wsv
变量 Jade <类型 = JadeView>
变量 主窗口ID <类型 = 整数>
```

> Receive-event methods (`JadeView_应用就绪`, etc.) can be generated by clicking the lightning icon next to the `Jade` variable — no hand-written signatures needed.

## IPC Replies & JSON Payload Parsing

```wsv
方法 IPC频道消息 <公开 类型 = 文本型 @强制输出 = 真 @输出名 = "ipcChannelMessageCallback">
参数 通道名 <类型 = 文本型>
参数 窗口ID <类型 = 整数>
参数 数据 <类型 = 文本型>
{
    变量 数据解构 <类型 = json_object>
    数据解构.创建自文本 (数据)

    如果 (通道名 == "ipc-echo")
    {
        返回 (数据)    // echo
    }
    如果 (通道名 == "set-theme")
    {
        // frontend payload: {"mode": "Dark"}
        Jade.窗口.置主题 (窗口ID, 数据解构.取文本 ("mode", "System"))
        返回 ("theme changed")
    }
    如果 (通道名 == "minimize")
    {
        Jade.窗口.最小化 (窗口ID)
        返回 ("minimized")
    }
    返回 ("unknown channel: " + 通道名)
}
```

## File Drag-Drop: Forward + Synchronous enter Rejection

```wsv
变量 拒绝拖入 <类型 = 逻辑型>

方法 JadeView_文件拖放 <接收事件 类型 = 整数 注释 = "Manual subscription required; enter/drop phases are interceptable.">
参数 来源对象 <类型 = JadeView>
参数 标记值 <类型 = 整数>
参数 窗口ID <类型 = 整数>
参数 数据 <类型 = 文本型>
{
    如果 (来源对象 == Jade)
    {
        // forward enter/over/drop/leave to the page (it stops receiving native drops
        // once the host takes over)
        Jade.发送IPC消息 (窗口ID, "drag-drop", 数据)

        如果真 (拒绝拖入)
        {
            变量 拖放数据 <类型 = json_object>
            如果真 (拖放数据.创建自文本 (数据))
            {
                如果真 (拖放数据.取文本 ("type", "") == "enter")
                {
                    返回 (1)   // reject at enter: cursor turns "no", no drop will follow
                }
            }
        }
    }
    返回 (0)
}
```

:::warning
Class member variables must not share a name with event parameters (the member above is `已注册热键ID`, not `热键ID`) — a clash produces a "duplicate name" compile error.
:::
